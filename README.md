## Overview
This project builds a behavioural analytics system to detect fake engagement 
on social media platforms. It differentiates between organic users and bots 
using behavioural pattern analysis and machine learning.

## Problem Statement
Social media platforms often experience fake engagement generated by bots or 
coordinated activity. This distorts genuine behavioural patterns and misleads 
content creators, advertisers, and platform algorithms.

## Dataset
- **Type:** Synthetic
- **Reason:** No real labeled bot behavioural dataset was available with 
  the required features
- **Generation Method:** Rule-based simulation using realistic distributions 
  derived from bot behaviour research
- **Records:** 1000 users (300 bots, 700 organic)
- **Features:** 9 behavioural features

## Features Engineered
| Feature | Description |
|--------|-------------|
| avg_posts_per_day | Average number of posts per day |
| posting_time_std | Standard deviation of posting times |
| engagement_burst_score | Frequency of sudden engagement spikes |
| followers_following_ratio | Ratio of followers to following |
| avg_comment_length | Average length of comments |
| unique_words_ratio | Ratio of unique words used |
| account_age_days | Age of the account in days |
| profile_completeness | How complete the profile is |
| night_activity_ratio | Ratio of activity during night hours |

## Model
- **Algorithm:** Random Forest Classifier
- **Accuracy:** 100%
- **ROC-AUC Score:** 1.0
- **Note:** Perfect scores are expected due to clearly defined synthetic 
  distributions. Real-world performance would vary.

## Output
- **Authenticity Score** (0-100)
- **Bot Probability** (0-1)
- **Risk Label** (Bot / Suspicious / Organic)

## How to Run

### 1. Install dependencies
```
pip install -r requirements.txt
```

### 2. Generate synthetic dataset
```
python data/generate_data.py
```

### 3. Train the model
```
python model/train_model.py
```

### 4. Launch the dashboard
```
python dashboard/app.py
```

Then open http://127.0.0.1:8050 in your browser.
